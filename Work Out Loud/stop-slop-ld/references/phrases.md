# Phrases to Cut: Learning Design Edition

## Throat-Clearing Openers

These delay the point. Cut them and start with the content.

| Phrase | Problem |
|--------|---------|
| "In this module, we will..." | Announces content instead of delivering it |
| "By the end of this lesson, you will be able to..." | Save for a stated objective, not narration filler |
| "Welcome to this course on..." | The learner knows where they are |
| "In this section, we'll explore..." | Cut. Start with what you're exploring. |
| "Before we begin, let's..." | You've already begun |
| "Let's start by looking at..." | Start looking. Drop the announcement. |
| "In today's learning, we'll cover..." | Cut entirely |
| "This is an important topic because..." | Show why it matters. Don't announce that it does. |

## Instructional Jargon

These words sound expert but say nothing specific.

| Word/Phrase | Replace with |
|-------------|--------------|
| "explore" | name the actual activity |
| "unpack" | "break down," or just say what's being examined |
| "dive into" | cut entirely or say what happens |
| "journey" | cut — learners aren't on a journey, they're doing a task |
| "navigate" | "handle," "work through," or name the specific action |
| "leverage" | "use" |
| "empower" | name what specifically becomes possible |
| "foster" | name what the person actually does |
| "cultivate" | same as above |
| "holistic" | cut or specify what's included |
| "robust" | cut or say what it actually does |
| "best practices" | name the specific practice |
| "skill set" | "skills" |
| "learning journey" | cut entirely |
| "transformative" | say what changes and for whom |
| "actionable" | if it's not obviously actionable, fix the content, not the adjective |
| "takeaways" | name the specific things |
| "key takeaways" | same. also, they're not "key" if everything is key |

## Objective-Writing Slop

These verbs appear in objectives but can't be observed or assessed.

| Verb | Problem | Replace with |
|------|---------|--------------|
| "Understand" | Unmeasurable. What does understanding look like? | Name the performance: "Explain," "Diagnose," "Decide," "Apply" |
| "Appreciate" | Unmeasurable. And slightly condescending. | Name what recognition or awareness looks like in action |
| "Gain awareness of" | Not a performance | "Identify," "Recognize," "Distinguish" |
| "Be familiar with" | Same problem | Name what familiarity produces |
| "Learn about" | That's the process, not the outcome | Name what they do with the knowledge |
| "Explore" | Not an outcome | Name the thing they can do after exploring |
| "Know" | Unmeasurable in isolation | Add: know *in order to do what*? |
| "Develop an understanding of" | Padded version of "understand" | Same fix |

## Encouragement Filler

These soften content for no reason. Learners don't need reassurance between sentences.

| Phrase | Problem |
|--------|---------|
| "Great question!" | Sycophantic. The question was already asked. |
| "Excellent work!" | Meaningless without specific feedback |
| "Don't worry — this is easier than it looks" | Condescending |
| "You've got this" | Empty |
| "Take your time with this one" | Unnecessary permission |
| "It's okay if this feels challenging" | The learner will decide how it feels |
| "Remember: there are no wrong answers" | Sometimes there are |
| "As you continue on your learning journey..." | Cut entirely |

## Transition Scaffolding

These exist for the writer's structure, not the learner's understanding.

| Phrase | Problem |
|--------|---------|
| "Now that we've covered X, let's move on to Y" | If the sequence is clear, this is redundant |
| "In the next section, you'll learn about..." | Cut. Let them get there. |
| "Let's recap what we've learned so far" | Only recap if the complexity requires it |
| "Before we wrap up..." | Just wrap up |
| "As we've seen..." | Cut. State the point. |
| "With that in mind..." | Cut. Make the connection explicit or don't. |
| "Building on what we just covered..." | Cut unless the connection isn't obvious |

## Meta-Commentary

Announcing what you're doing instead of doing it.

| Phrase | Problem |
|--------|---------|
| "This is a critical concept" | Show it, don't announce it |
| "Pay close attention to this part" | Everything should deserve attention |
| "This is worth noting" | Then note it without the flag |
| "It's important to remember that..." | Cut and state the thing directly |
| "Keep in mind that..." | Cut |
| "Note that..." | Usually cuttable |
| "It's worth mentioning that..." | Cut the throat-clearing |
| "As you can see..." | They can see. Remove. |
